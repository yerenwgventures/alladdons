# Duplicate Product BOM

## Overview
Duplicating a product will duplicate the products 
    BOM also

## Installation

### Prerequisites
**Required Odoo Modules:**
- sale

**Custom Dependencies:**
- mrp (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `duplicate_product_bom` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Duplicate Product BOM"
6. Click Install
