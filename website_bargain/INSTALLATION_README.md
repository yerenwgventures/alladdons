# Product Bidding In ECommerce

## Overview
App To Add Bidding option In Website

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_website_sale (ensure this module is available)
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `website_bargain` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Product Bidding In ECommerce"
6. Click Install
