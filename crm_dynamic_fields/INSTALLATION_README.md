# CRM Dynamic Fields

## Overview
Helps to Add Custom Fields for Opportunity from user interface

## Installation

### Prerequisites
**Required Odoo Modules:**
- crm

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `crm_dynamic_fields` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "CRM Dynamic Fields"
6. Click Install
