# Invoice Multi Level Approval

## Overview
This module add the multiple approval option for invoice,
     bill, refund and credit notes with single or multiple approvals

## Installation

### Prerequisites
**Required Odoo Modules:**
- account

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `invoice_multi_approval` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Invoice Multi Level Approval"
6. Click Install
