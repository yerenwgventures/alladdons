# Automatic Invoice And Post

## Overview
 Auto Invoice Generation and Auto Sending of Invoice on 
     Delivery validation.

## Installation

### Prerequisites
**Required Odoo Modules:**
- stock
- account

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `automatic_invoice_and_post` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Automatic Invoice And Post"
6. Click Install
