# Age Restricted Products POS

## Overview
This Module will help to restrict the age restricted products in pos.

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_point_of_sale (ensure this module is available)
- product (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `age_restricted_product_pos` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Age Restricted Products POS"
6. Click Install
