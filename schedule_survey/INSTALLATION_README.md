# Survey Scheduler

## Overview
This module helps you schedule the survey, and the survey will be automatically sent to the participants.

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_survey (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `schedule_survey` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Survey Scheduler"
6. Click Install

## Compatibility

### Odoo Version
- **Supported:** Odoo 18.0
- **Version:** 18.0.1.0.0
