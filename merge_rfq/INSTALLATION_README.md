# Merge RFQ

## Overview
This module merege two or more RFQ by cancelling or deleting
        the others in RFQ and RFQ sent state.

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `merge_rfq` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Merge RFQ"
6. Click Install
