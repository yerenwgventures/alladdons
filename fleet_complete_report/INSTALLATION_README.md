### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `fleet_complete_report` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Fleet Complete Report"
6. Click Install
