## Installation

### Prerequisites
**Required Odoo Modules:**
- account
- website
- sale

**Custom Dependencies:**
- stub_event (ensure this module is available)
- fleet (ensure this module is available)
- stub_point_of_sale (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `hotel_management_odoo` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Hotel Management"
6. Click Install
