# Import Employee/Partner Image

## Overview
Import Partner/Employee Image From Url and Local storage using CSV and XLSX Files

## Installation

### Prerequisites
**Required Odoo Modules:**
- hr

**Custom Dependencies:**
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `import_partner_employee_image` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Import Employee/Partner Image"
6. Click Install
