# Franchise Management

## Overview
This Module will help to manage franchisees and dealers.

## Installation

### Prerequisites
**Required Odoo Modules:**
- account
- sale

**Custom Dependencies:**
- product (ensure this module is available)
- stub_website_sale (ensure this module is available)
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `franchise_management` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Franchise Management"
6. Click Install
