# Batch Delivery Tracking

## Overview
It Helps to Track Multiple Delivery Orders in a Batch Transfer

## Installation

### Prerequisites
**Custom Dependencies:**
- stock_delivery (ensure this module is available)
- stock_picking_batch (ensure this module is available)
- stub_delivery (ensure this module is available)
- stub_sale_stock (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `batch_delivery_tracking` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Batch Delivery Tracking"
6. Click Install
