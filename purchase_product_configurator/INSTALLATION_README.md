# Purchase Product Configurator

## Overview
Purchase variant selection options for products.

## Installation

### Prerequisites
**Custom Dependencies:**
- purchase_product_matrix (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `purchase_product_configurator` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Purchase Product Configurator"
6. Click Install
