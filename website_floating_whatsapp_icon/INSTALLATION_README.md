# WhatsApp Floating Icon in Website

## Overview
Adds a WhatsApp Floating Icon in Website to connect with
    WhatsApp web.

## Installation

### Prerequisites
**Required Odoo Modules:**
- website

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `website_floating_whatsapp_icon` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "WhatsApp Floating Icon in Website"
6. Click Install
