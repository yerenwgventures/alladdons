## Installation

### Prerequisites

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `dynamic_link_snippet` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "URL Snippet"
6. Click Install
