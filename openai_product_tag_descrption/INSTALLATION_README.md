### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `openai_product_tag_descrption` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Generate Product Description/Tags Using OpenAI"
6. Click Install
