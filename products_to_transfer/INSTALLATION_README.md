# Add Multiple Products to Inventory Transfer

## Overview
Easily add multiple products to a picking directly from the product list view with this new feature

## Installation

### Prerequisites
**Required Odoo Modules:**
- stock

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `products_to_transfer` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Add Multiple Products to Inventory Transfer"
6. Click Install
