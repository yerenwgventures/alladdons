## Installation

### Prerequisites
**Required Odoo Modules:**
- sale
- account
- website

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `whatsapp_mail_messaging` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Odoo Whatsapp Connector"
6. Click Install
