# Costing Method: Last Purchase Price

## Overview
Introducing new costing method in Odoo 'Last Purchase Price'

## Installation

### Prerequisites
**Custom Dependencies:**
- stock_account (ensure this module is available)
- purchase_stock (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `stock_last_purchase_price` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Costing Method: Last Purchase Price"
6. Click Install
