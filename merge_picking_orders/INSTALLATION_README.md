# Merge Picking Orders

## Overview
With This Module, You Can Easily Merge Pickings such as Receipts, Delivery Orders and Returns

## Installation

### Prerequisites
**Required Odoo Modules:**
- stock

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `merge_picking_orders` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Merge Picking Orders"
6. Click Install
