## Installation

### Prerequisites
**Required Odoo Modules:**
- account

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `sale_discount_total` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Sale Discount on Total Amount"
6. Click Install
