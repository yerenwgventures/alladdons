### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `advance_cash_flow_statements` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Advanced Cash Flow Statements"
6. Click Install
