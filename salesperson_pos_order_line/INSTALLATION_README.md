1. Ensure all prerequisites are installed
2. Copy the `salesperson_pos_order_line` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Sales Person On POS Order Line"
6. Click Install
