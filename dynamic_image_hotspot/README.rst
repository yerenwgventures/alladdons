Dynamic Image Hotspot
======================
The module allows users to add a hotspot for snippet images. A product link can be further added
to the hotspot so that whenever a user clicks on the hotspot they are directed to the
product page in website shop.

=============
* Easy setup through Odoo Apps interface
* No complex configuration required
