# Theme The Chef

## Overview
Theme The Chef is a popular attractive and unique front end theme for your restaurant website.

## Installation

### Prerequisites
**Required Odoo Modules:**
- website

**Custom Dependencies:**
- mass_mailing (ensure this module is available)
- website_sale_wishlist (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `theme_the_chef` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Theme The Chef"
6. Click Install
