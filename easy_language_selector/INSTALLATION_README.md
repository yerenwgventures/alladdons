# Easy Language Selector

## Overview
User can change the language of the odoo back-end in easy way

## Installation

### Prerequisites
### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `easy_language_selector` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Easy Language Selector"
6. Click Install
