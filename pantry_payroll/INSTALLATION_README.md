# Payroll Management In Pantry

## Overview
Efficiently oversee payroll operations by factoring in 
     pantry expenditures through seamless deductions

## Installation

### Prerequisites
**Required Odoo Modules:**
- hr

**Custom Dependencies:**
- sale_management (ensure this module is available)
- stub_hr_payroll (ensure this module is available)
- stub_hr_contract (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `pantry_payroll` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Payroll Management In Pantry"
6. Click Install
