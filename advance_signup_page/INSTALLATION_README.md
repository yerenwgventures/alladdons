# Advance Signup Page

## Overview
The user can design a more creative and distinctive sign-up pagefor their Odoo website by using Odoo Advance Sign Up

## Installation

### Prerequisites
**Required Odoo Modules:**
- website

**Custom Dependencies:**
- auth_signup (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `advance_signup_page` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Advance Signup Page"
6. Click Install

## Compatibility

### Odoo Version
- **Supported:** Odoo 18.0
- **Version:** 18.0.1.0.0

### Potential Conflicts
