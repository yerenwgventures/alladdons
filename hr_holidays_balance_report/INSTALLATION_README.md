# HR Balance Leave Report

## Overview
Allocated balance, taken leaves and remaining balance per leave type for each employee

## Installation

### Prerequisites
**Custom Dependencies:**
- hr_holidays (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `hr_holidays_balance_report` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "HR Balance Leave Report"
6. Click Install
