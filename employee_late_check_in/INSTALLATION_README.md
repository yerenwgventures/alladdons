# Employee Late Check-in

## Overview
Employee Late Check-in module for tracking and managing late check-ins of employees and may deduct salary from payslip

## Installation

### Prerequisites
**Required Odoo Modules:**
- hr

**Custom Dependencies:**
- hr_attendance (ensure this module is available)
- stub_hr_contract (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `employee_late_check_in` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Employee Late Check-in"
6. Click Install

## Compatibility

### Odoo Version
- **Supported:** Odoo 18.0
- **Version:** 18.0.1.0.0

### Potential Conflicts
