# Employee Attendance Xlsx Report

## Overview
This module will manage  the attendance report of employees 
    in xlsx

## Installation

### Prerequisites
**Custom Dependencies:**
- hr_attendance (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `hr_attendance_xlsx_report` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Employee Attendance Xlsx Report"
6. Click Install
