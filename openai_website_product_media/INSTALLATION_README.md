# OpenAI Ecommerce Product Media

## Overview
Open AI Product Media Images for Ecommerce.

## Installation

### Prerequisites
**Required Odoo Modules:**
- stock

**Custom Dependencies:**
- product (ensure this module is available)
- openai_odoo_base (ensure this module is available)
- stub_website_sale (ensure this module is available)
- openai_product_images (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `openai_website_product_media` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "OpenAI Ecommerce Product Media"
6. Click Install
