# Disable Variants in Website

## Overview
This module is used to hide variants in the website

## Installation

### Prerequisites
**Custom Dependencies:**
- product (ensure this module is available)
- stub_website_sale (ensure this module is available)
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `website_hide_variants` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Disable Variants in Website"
6. Click Install
