### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `odoo_selection_field_configurator` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Odoo Selection Field Configurator"
6. Click Install
