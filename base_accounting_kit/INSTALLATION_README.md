## Installation

### Prerequisites
**Required Odoo Modules:**
- account
- sale

**Custom Dependencies:**
- account_check_printing (ensure this module is available)
- analytic (ensure this module is available)
- base_account_budget (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `base_accounting_kit` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Odoo 18 Full Accounting Kit for Community"
6. Click Install
