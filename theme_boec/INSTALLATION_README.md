# Theme Boec

## Overview
Theme Boec is an attractive and modern eCommerce Website theme

## Installation

### Prerequisites
**Custom Dependencies:**
- website_blog (ensure this module is available)
- website_sale_wishlist (ensure this module is available)
- stub_website_sale (ensure this module is available)
- website_sale_comparison (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `theme_boec` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Theme Boec"
6. Click Install
