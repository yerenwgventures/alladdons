# File Upload In Survey

## Overview
Attachment of File in Survey Form

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_survey (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `survey_upload_file` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "File Upload In Survey"
6. Click Install
