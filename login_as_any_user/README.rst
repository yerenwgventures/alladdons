Login As Any User
=================
The Login As Any User module in Odoo 18 enables administrators to log in as any
user in the system. With this module, you can test user access and permissions,
troubleshoot issues, or view the system as a specific user.

Configuration
=============
* Easy setup through Odoo Apps interface
* No complex configuration required
