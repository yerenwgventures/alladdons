# Project and Task Progress Status

## Overview
Keep your finger on the projects pulse with this module. Featuring a dynamic progress bar, it provides a visual snapshotof project and task advancement. Toggle effortlessly betweenKanban and list views for a personalized project tracking experience.

## Installation

### Prerequisites
**Required Odoo Modules:**
- project

**Custom Dependencies:**
- hr_timesheet (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `project_progress_bar` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Project and Task Progress Status"
6. Click Install
