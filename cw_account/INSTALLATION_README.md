## Installation

### Prerequisites
**Required Odoo Modules:**
- account

**Custom Dependencies:**
- cw_stock (ensure this module is available)
- uom (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `cw_account` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Catch Weight Management: Accounting"
6. Click Install
