# One2many Excel Report

## Overview
One2many Excel Report which is used to print the excel report 
    of one2many fields.

## Installation

### Prerequisites
### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `one2many_excel_report` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "One2many Excel Report"
6. Click Install
