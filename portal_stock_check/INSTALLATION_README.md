## Overview

## Installation

### Prerequisites

- stock

**Custom Dependencies:**
- portal (ensure this module is available)

- stub_website_sale (ensure this module is available)
- product (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `portal_stock_check` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Portal Product Availability"
6. Click Install
