# Sale Order Payment Status

## Overview
Displays the payment status and details in the Sale Order.

## Installation

### Prerequisites
**Required Odoo Modules:**
- account

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `payment_status_in_sale` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Sale Order Payment Status"
6. Click Install
