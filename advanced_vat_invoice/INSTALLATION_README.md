# E-Invoicing For Saudi | Saudi VAT Invoice | Saudi Electronic Invoice | Saudi Zatca

## Overview
This module enables VAT-compliant e-Invoicing for Saudi Arabia, following ZATCA (Fatoorah) regulations. It supports generating electronic invoices with QR codes and structured formats for both simplified and standard invoices

## Installation

### Prerequisites
**Required Odoo Modules:**
- account

**Custom Dependencies:**
- account_payment (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `advanced_vat_invoice` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "E-Invoicing For Saudi | Saudi VAT Invoice | Saudi Electronic Invoice | Saudi Zatca"
6. Click Install
