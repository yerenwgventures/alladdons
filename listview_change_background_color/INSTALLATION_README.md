# Listview Change Background Color

## Overview
Change Background Colors Of Most Priority Records In Tree or 
     List View

## Installation

### Prerequisites
**Required Odoo Modules:**
- crm
- sale

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `listview_change_background_color` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Listview Change Background Color"
6. Click Install
