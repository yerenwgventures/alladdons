# Customer Credit Payment In Website

## Overview
Assign the credit amount from the website and the sales

## Installation

### Prerequisites
**Required Odoo Modules:**
- purchase

**Custom Dependencies:**
- sale_management (ensure this module is available)
- product (ensure this module is available)
- stub_website_sale (ensure this module is available)
- payment (ensure this module is available)
- payment_demo (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `customer_credit_payment_website` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Customer Credit Payment In Website"
6. Click Install
