# Odoo Vendor Portal

## Installation

### Prerequisites
**Required Odoo Modules:**
- website
- purchase
- stock

**Custom Dependencies:**
- portal (ensure this module is available)
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `vendor_portal_odoo` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Odoo Vendor Portal"
6. Click Install
