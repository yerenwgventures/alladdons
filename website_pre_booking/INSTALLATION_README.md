# Website Pre Booking

## Overview
Allows pre booking option for website

## Installation

### Prerequisites
**Custom Dependencies:**
- portal (ensure this module is available)
- stub_website_sale (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `website_pre_booking` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Website Pre Booking"
6. Click Install
