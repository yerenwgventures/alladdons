## Installation

### Prerequisites
**Required Odoo Modules:**
- website

**Custom Dependencies:**
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `website_gdpr_odoo` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Website GDPR In Odoo"
6. Click Install
