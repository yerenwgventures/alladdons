# Global Search

## Overview
Easy Search in Customers, Products, Sale, Purchase, Inventory
    and Accounting modules

## Installation

### Prerequisites
**Required Odoo Modules:**
- stock
- sale
- purchase

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `master_search` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Global Search"
6. Click Install
