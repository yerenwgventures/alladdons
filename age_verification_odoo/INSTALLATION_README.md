# Age Verification in Website Login

## Overview
This module acts as a gate,allowing only users over the
     specified age range to access the platform.

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_website_sale (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `age_verification_odoo` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Age Verification in Website Login"
6. Click Install
