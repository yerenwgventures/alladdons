### Prerequisites
**Custom Dependencies:**
- website_sale_wishlist (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `wishlist_product_website_backend` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Wishlist Products In Backend"
6. Click Install
