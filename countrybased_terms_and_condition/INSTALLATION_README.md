# Country-Based Terms and Condition

## Overview
Customise your terms and condition based on  the country 
     of the contacts

## Installation

### Prerequisites
**Custom Dependencies:**
- sale_management (ensure this module is available)
- contacts (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `countrybased_terms_and_condition` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Country-Based Terms and Condition"
6. Click Install
