# Accounting Dashboard Odoo18

## Overview
The Odoo Accounting Dashboard shows key financial metrics, 
     tracks payments, and visualizes income and expenses in one place.

## Installation

### Prerequisites
**Required Odoo Modules:**
- account

**Custom Dependencies:**
- base_accounting_kit (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `odoo_accounting_dashboard` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Accounting Dashboard Odoo18"
6. Click Install
