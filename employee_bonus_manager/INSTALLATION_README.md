## Installation

### Prerequisites
**Required Odoo Modules:**
- account
- hr

**Custom Dependencies:**
- stub_hr_payroll (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `employee_bonus_manager` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Employee Bonus"
6. Click Install
