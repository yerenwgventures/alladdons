# Controlled Point Of Sale

## Overview
Manager Can Control Discount & Price For the POS.

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_point_of_sale (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `pos_controlled_interface` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Controlled Point Of Sale"
6. Click Install
