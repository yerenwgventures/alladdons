# POS Idle Session Lock

## Overview
The Module Allows the POS User to Set Screen Lock for POS 
    Screen

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_point_of_sale (ensure this module is available)
- pos_hr (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `pos_idle_time_session_lock` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "POS Idle Session Lock"
6. Click Install
