# SignUp With Mobile

## Overview
A Module For SignUp With Mobile Number.

## Installation

### Prerequisites
**Custom Dependencies:**
- portal (ensure this module is available)
- auth_signup (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `signup_with_twilio` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "SignUp With Mobile"
6. Click Install
