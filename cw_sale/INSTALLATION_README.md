# Catch Weight - Sale

## Overview
Enable Catch Weight Management System in Sales Module

## Installation

### Prerequisites
**Custom Dependencies:**
- sale_management (ensure this module is available)
- stub_sale_stock (ensure this module is available)
- cw_stock (ensure this module is available)
- uom (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `cw_sale` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Catch Weight - Sale"
6. Click Install
