# Daily/Weekly Task Status Report

## Overview
Automatic daily and weekly report.

## Installation

### Prerequisites
**Custom Dependencies:**
- analytic (ensure this module is available)
- hr_timesheet (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `auto_daily_weekly_report` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Daily/Weekly Task Status Report"
6. Click Install
