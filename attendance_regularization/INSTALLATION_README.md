# Open HRMS Attendance Regularization

## Overview
Assigning Attendance for the Employees with Onsight Jobs

## Installation

### Prerequisites
**Required Odoo Modules:**
- hr

**Custom Dependencies:**
- hr_attendance (ensure this module is available)
- hr_holidays (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `attendance_regularization` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Open HRMS Attendance Regularization"
6. Click Install
