### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `cts_theme_rozz` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Theme Rozz"
6. Click Install
