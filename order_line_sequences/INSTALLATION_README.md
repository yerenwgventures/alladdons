# Order Line Sequences/Line Numbers

## Overview
Sequence numbers in order lines of sales,purchase and delivery.

## Installation

### Prerequisites
**Required Odoo Modules:**
- purchase
- stock

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `order_line_sequences` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Order Line Sequences/Line Numbers"
6. Click Install
