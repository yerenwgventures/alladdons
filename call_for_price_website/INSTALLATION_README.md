# Website Call For Price

## Overview
Helps to hide price of product from shop

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_website_sale (ensure this module is available)
- website_sale_stock (ensure this module is available)
- website_sale_wishlist (ensure this module is available)
- website_sale_comparison (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `call_for_price_website` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Website Call For Price"
6. Click Install
