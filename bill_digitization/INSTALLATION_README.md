# Bill Digitization

## Overview
Converting traditional paper-based bills into digital 
     formats.

## Installation

### Prerequisites
**Custom Dependencies:**
- base_accounting_kit (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `bill_digitization` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Bill Digitization"
6. Click Install
