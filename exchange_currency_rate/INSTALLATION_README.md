# Manual Currency Exchange Rate

## Overview
By using this module ,we can change the currency rate manually
     in sale ,purchase and invoice. 

## Installation

### Prerequisites
**Required Odoo Modules:**
- purchase
- account

**Custom Dependencies:**
- sale_management (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `exchange_currency_rate` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Manual Currency Exchange Rate"
6. Click Install
