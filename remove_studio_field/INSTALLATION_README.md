# Remove Studio Fields

## Overview
The module helps to remove fields added from Studio

## Installation

### Prerequisites
**Custom Dependencies:**
- web_studio (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `remove_studio_field` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Remove Studio Fields"
6. Click Install

## Compatibility

### Odoo Version

- **Version:** 18.0.1.0.0
