# Beauty Spa Management

## Overview
Beauty Parlour Management with Online Booking System

## Installation

### Prerequisites
**Required Odoo Modules:**
- account
- website

**Custom Dependencies:**
- base_setup (ensure this module is available)
- contacts (ensure this module is available)
- stub_point_of_sale (ensure this module is available)
- calendar (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `salon_management` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "Beauty Spa Management"
6. Click Install
