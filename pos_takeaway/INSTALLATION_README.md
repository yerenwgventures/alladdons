# POS Restaurant Dine-in/TakeAway

## Overview
This module will add the options Dine-in and Take away in Odoo POS.

## Installation

### Prerequisites
**Custom Dependencies:**
- stub_point_of_sale (ensure this module is available)
- pos_restaurant (ensure this module is available)

### Installation Steps
1. Ensure all prerequisites are installed
2. Copy the `pos_takeaway` folder to your Odoo addons directory
3. Restart Odoo server
4. Update apps list: Settings → Apps → Update Apps List
5. Search for "POS Restaurant Dine-in/TakeAway"
6. Click Install
